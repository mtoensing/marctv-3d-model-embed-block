# Minimal Model Viewer Block

Minimaler Gutenberg-Block fuer `GLB`- und `glTF`-Modelle.

## Eigenschaften

- lokales `model-viewer`-Asset statt CDN-Zwang
- dynamischer Block mit PHP-Rendercallback
- Editor-Vorschau direkt im Block
- `glb` und `gltf` werden fuer die Mediathek freigegeben

## Nutzung

1. Plugin aktivieren
2. GLB-Datei in die Mediathek laden
3. Block `3D Model Viewer` in einen Beitrag einfuegen
4. GLB-URL eintragen oder Datei aus der Mediathek waehlen
